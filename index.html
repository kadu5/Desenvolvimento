Explicação do que é html semântico

HTML semântico é uma abordagem para escrever código HTML que atribui significado e estruturação adequada aos elementos do documento, tornando-o mais legível para desenvolvedores, troca de busca
e também acessível para pessoas com deficiência visual ou com tecnologias assistivas. A ideia por trás do HTML semântico é usar elementos HTML que expressam claramente o conteúdo que eles contêm, ao invés de usar apenas elementos genéricos que são estilizados posteriormente com CSS. Antes de entender o que é HTML semântico, é importante entender o que é "semântica". Semântica refere-se ao significado ou sentido do texto ou informação. Em relação ao HTML, a semântica é a recepção de significado aos elementos utilizados na marcação do conteúdo da página.

Explicação do que é CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação e layout de documentos HTML (ou XML). Ele permite que você aplique estilos, como cores, fontes, margens, espaçamentos e posicionamento, para controlar a aparência visual e a formatação de uma página da web ou um documento arquitetônico. A principal função do CSS é separar o conteúdo do documento (definido em HTML) da sua apresentação (definida em CSS). Dessa forma, você pode criar um arquivo HTML com a estrutura lógica do conteúdo e um arquivo CSS separado para controlar como esse conteúdo deve ser apresentado aos usuários.
